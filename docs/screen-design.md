# 画面設計書

## 1. 画面一覧

| # | 画面名 | URL | 認証必須 |
|---|--------|-----|----------|
| 1 | サインアップ画面 | `/signup` | 不要 |
| 2 | ログイン画面 | `/login` | 不要 |
| 3 | タイムライン画面 | `/` | 必要 |
| 4 | 投稿詳細画面 | `/posts/:id` | 必要 |
| 5 | 投稿作成画面 | `/posts/new` | 必要 |
| 6 | プロフィール画面 | `/users/:id` | 必要 |
| 7 | プロフィール編集画面 | `/users/:id/edit` | 必要（本人のみ） |
| 8 | フォロー一覧画面 | `/users/:id/following` | 必要 |
| 9 | フォロワー一覧画面 | `/users/:id/followers` | 必要 |
| 10 | ユーザー検索画面 | `/users/search` | 必要 |

## 2. 画面遷移図

```
[サインアップ画面 /signup]
  ├── 登録成功 → [タイムライン画面 /]
  └── ログインリンク → [ログイン画面 /login]

[ログイン画面 /login]
  ├── ログイン成功 → [タイムライン画面 /]
  └── 新規登録リンク → [サインアップ画面 /signup]

[タイムライン画面 /]（ホーム）
  ├── [全体] / [フォロー中] タブ切り替え（ページ内）
  ├── 投稿作成ボタン → [投稿作成画面 /posts/new]
  ├── 投稿カードクリック → [投稿詳細画面 /posts/:id]
  ├── ユーザー名クリック → [プロフィール画面 /users/:id]
  ├── 検索アイコンクリック → [ユーザー検索画面 /users/search]
  └── ログアウト → [ログイン画面 /login]

[投稿作成画面 /posts/new]
  ├── 投稿成功 → [タイムライン画面 /]
  └── キャンセル → [タイムライン画面 /]

[投稿詳細画面 /posts/:id]
  ├── コメント投稿（その場で更新）
  ├── いいねボタン（その場で更新）
  ├── 投稿者ユーザー名クリック → [プロフィール画面 /users/:id]
  ├── コメント投稿者ユーザー名クリック → [プロフィール画面 /users/:id]
  └── 戻るボタン → [タイムライン画面 /]

[プロフィール画面 /users/:id]
  ├── フォローボタン（その場で更新）
  ├── フォロー数クリック → [フォロー一覧画面 /users/:id/following]
  ├── フォロワー数クリック → [フォロワー一覧画面 /users/:id/followers]
  ├── 編集ボタン（本人のみ） → [プロフィール編集画面 /users/:id/edit]
  ├── 投稿カードクリック → [投稿詳細画面 /posts/:id]
  └── 戻るボタン → [タイムライン画面 /]

[プロフィール編集画面 /users/:id/edit]
  ├── 保存成功 → [プロフィール画面 /users/:id]
  └── キャンセル → [プロフィール画面 /users/:id]

[フォロー一覧画面 /users/:id/following]
  ├── ユーザー名クリック → [プロフィール画面 /users/:id]
  └── 戻るボタン → [プロフィール画面 /users/:id]

[フォロワー一覧画面 /users/:id/followers]
  ├── ユーザー名クリック → [プロフィール画面 /users/:id]
  └── 戻るボタン → [プロフィール画面 /users/:id]

[ユーザー検索画面 /users/search]
  ├── 検索欄にキーワード入力 → 検索結果をリアルタイム表示
  ├── フォローボタン（その場で更新）
  ├── ユーザー名クリック → [プロフィール画面 /users/:id]
  └── 戻るボタン → [タイムライン画面 /]
```

## 3. 各画面の詳細

### 3.1 サインアップ画面（`/signup`）

**目的**: 新規ユーザーのアカウント作成

**ワイヤーフレーム**

```
+--------------------------------------------+
|           RaiseTimeLine                    |
+--------------------------------------------+
|  ユーザー名  [________________]            |
|  メールアドレス [________________]         |
|  パスワード  [________________]            |
|  パスワード確認 [________________]         |
|                                            |
|          [  アカウント登録  ]              |
|                                            |
|  すでにアカウントをお持ちの方は ログイン   |
+--------------------------------------------+
```

**UI要素**

| 要素 | 種別 | バリデーション |
|------|------|--------------|
| ユーザー名 | テキスト入力 | 必須・50文字以内 |
| メールアドレス | メール入力 | 必須・メール形式 |
| パスワード | パスワード入力 | 必須・8文字以上 |
| パスワード確認 | パスワード入力 | 必須・パスワードと一致 |
| アカウント登録ボタン | ボタン | バリデーション後に送信 |
| ログインへのリンク | テキストリンク | 既存ユーザー向け |

---

### 3.2 ログイン画面（`/login`）

**目的**: 既存ユーザーの認証

**ワイヤーフレーム**

```
+--------------------------------------------+
|           RaiseTimeLine                    |
+--------------------------------------------+
|  メールアドレス [________________]         |
|  パスワード  [________________]            |
|                                            |
|            [  ログイン  ]                  |
|                                            |
|  アカウントをお持ちでない方は 新規登録     |
+--------------------------------------------+
```

**UI要素**

| 要素 | 種別 | バリデーション |
|------|------|--------------|
| メールアドレス | メール入力 | 必須 |
| パスワード | パスワード入力 | 必須 |
| ログインボタン | ボタン | |
| 新規登録へのリンク | テキストリンク | |

---

### 3.3 タイムライン画面（`/`）

**目的**: 全体タイムラインまたはフォロータイムラインを閲覧する

**ワイヤーフレーム**

```
+--------------------------------------------+
| ロゴ              🔍  ユーザー名  ログアウト|
+============================================+
|  [ 全体 ]  [ フォロー中 ]                  |
+--------------------------------------------+
|                    [ 投稿する ]            |
+--------------------------------------------+
|  ┌──────────────────────────────────────┐  |
|  │ [アイコン] ユーザー名    2025/06/08  │  |
|  │ 投稿テキスト本文                      │  |
|  │ [画像1][画像2][画像3][画像4]          │  |
|  │ ♡ 12        💬 3                     │  |
|  └──────────────────────────────────────┘  |
|  ┌──────────────────────────────────────┐  |
|  │ ...（繰り返し）                       │  |
|  └──────────────────────────────────────┘  |
+--------------------------------------------+
```

**UI要素**

| 要素 | 種別 | 備考 |
|------|------|------|
| ヘッダー | 固定ナビゲーション | ロゴ・検索アイコン・ユーザー名・ログアウトボタン |
| タイムラインタブ | タブコンポーネント | 全体／フォロー中を切り替え（デフォルト：全体） |
| 投稿するボタン | ボタン | 投稿作成画面へ遷移 |
| 投稿カード | カードコンポーネント | クリックで投稿詳細画面へ |
| ユーザー名（カード内） | リンク | クリックでプロフィール画面へ遷移 |
| いいねボタン | トグルボタン | カード上でいいね可能 |
| コメント数 | テキスト表示 | クリックで投稿詳細画面へ |

---

### 3.4 投稿作成画面（`/posts/new`）

**目的**: テキストと画像を含む投稿を作成する

**ワイヤーフレーム**

```
+--------------------------------------------+
| ← 戻る           投稿を作成               |
+--------------------------------------------+
|  [アイコン]                                |
|  +--------------------------------------+  |
|  | いまどんなことを考えていますか？      |  |
|  |                                      |  |
|  +--------------------------------------+  |
|                              残り 280文字  |
|                                            |
|  [画像1] [画像2] [画像3] [画像4]          |
|  [📷 画像を追加]                           |
|                                            |
|  [キャンセル]              [  投稿する  ] |
+--------------------------------------------+
```

**UI要素**

| 要素 | 種別 | 備考 |
|------|------|------|
| テキストエリア | 多行テキスト入力 | 必須・残り文字数カウント表示（280文字上限） |
| 画像添付ボタン | ファイル選択 | 最大4枚・複数選択可・JPEG/PNG/GIF |
| 画像プレビュー | サムネイル表示 | 個別削除可能 |
| 投稿するボタン | ボタン | バリデーション後に送信 |
| キャンセルボタン | ボタン | タイムライン画面へ戻る |

---

### 3.5 投稿詳細画面（`/posts/:id`）

**目的**: 投稿の全文・画像・コメントを表示し、いいね・コメントを行う

**ワイヤーフレーム**

```
+--------------------------------------------+
| ← 戻る                                    |
+--------------------------------------------+
| [アイコン] ユーザー名*        2025/06/08  |
| 投稿テキスト本文（全文）                   |
| [画像1][画像2][画像3][画像4]               |
|                                            |
| ♡ 12  [いいねする]     💬 3              |
+--------------------------------------------+
| コメント（3件）                            |
|  [アイコン] ユーザー名*  コメント本文 [削除]|
|  [アイコン] ユーザー名*  コメント本文      |
+--------------------------------------------+
| [アイコン] [コメントを入力...]  [送信]     |
+--------------------------------------------+
  * ユーザー名クリック → プロフィール画面へ遷移
```

---

### 3.6 プロフィール画面（`/users/:id`）

**目的**: 特定ユーザーの情報とその投稿一覧を表示する

**ワイヤーフレーム**

```
+--------------------------------------------+
| ← 戻る                                    |
+--------------------------------------------+
|       [アイコン画像]                       |
|       ユーザー名          [フォロー]       |
|                        （本人：[編集]）    |
|       自己紹介文                           |
|       フォロー: 12  フォロワー: 34         |
+--------------------------------------------+
| 投稿一覧                                   |
|  ┌──────────────────────────────────────┐  |
|  │ 投稿カード...                        │  |
|  └──────────────────────────────────────┘  |
+--------------------------------------------+
```

**UI要素**

| 要素 | 種別 | 備考 |
|------|------|------|
| アイコン画像 | 画像表示 | S3からロード |
| ユーザー名 | テキスト表示 | |
| 自己紹介文 | テキスト表示 | |
| フォロー数 | テキストリンク | クリックでフォロー一覧へ |
| フォロワー数 | テキストリンク | クリックでフォロワー一覧へ |
| フォローボタン | トグルボタン | 他ユーザーのみ表示 |
| 編集ボタン | ボタン | 本人のみ表示 |
| 投稿一覧 | カードリスト | そのユーザーの投稿のみ |

---

### 3.7 プロフィール編集画面（`/users/:id/edit`）

**目的**: 自分のプロフィール情報を更新する

**UI要素**

| 要素 | 種別 | 備考 |
|------|------|------|
| アイコン画像プレビュー | 画像表示 | 現在のアイコンを表示 |
| アイコン変更ボタン | ファイル選択 | JPEG/PNG・5MB以内・S3にアップロード |
| ユーザー名 | テキスト入力 | 必須・50文字以内 |
| 自己紹介文 | 多行テキスト入力 | 任意・160文字以内 |
| 保存ボタン | ボタン | |
| キャンセルボタン | ボタン | プロフィール画面へ戻る |

### 3.10 ユーザー検索画面（`/users/search`）

**目的**: ユーザー名の部分一致検索で他ユーザーを探し、フォローする

**ワイヤーフレーム**

```
+--------------------------------------------+
| ← 戻る          ユーザーを検索            |
+--------------------------------------------+
| 🔍 [ユーザー名を入力...]                  |
+--------------------------------------------+
|  ┌──────────────────────────────────────┐  |
|  │ [アイコン] ユーザー名    [フォロー]   │  |
|  └──────────────────────────────────────┘  |
|  ┌──────────────────────────────────────┐  |
|  │ ...（繰り返し）                       │  |
|  └──────────────────────────────────────┘  |
+--------------------------------------------+
```

**UI要素**

| 要素 | 種別 | 備考 |
|------|------|------|
| 検索欄 | テキスト入力 | デバウンス（約300ms）でリアルタイム検索 |
| ユーザーカード（繰り返し） | カード | アイコン・ユーザー名・フォローボタン |
| フォローボタン | トグルボタン | フォロー中/未フォローで表示切り替え |

**検索仕様**
- ユーザー名の部分一致（大文字・小文字を区別しない）
- 自分自身は表示しない
- 最大20件表示
- 未入力・空白の場合は結果を表示しない
