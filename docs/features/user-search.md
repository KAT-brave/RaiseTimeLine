# 機能定義書：ユーザー検索

## 概要

ユーザー名の部分一致検索で他のユーザーを検索できる。検索結果からそのままフォロー操作が可能。また、投稿・コメントのユーザー名をクリックしてプロフィール画面からもフォローできる。

---

## 機能一覧

| # | 機能 | 説明 |
|---|------|------|
| 1 | ユーザー名検索 | ユーザー名の部分一致検索でユーザーを絞り込む |
| 2 | 検索結果からフォロー | 検索結果一覧のフォローボタンからその場でフォローできる |
| 3 | ユーザー名クリックでプロフィール遷移 | 投稿・コメント・検索結果のユーザー名クリックでプロフィール画面へ遷移しフォローできる |

---

## 詳細仕様

### ユーザー名検索

**検索方式**: ユーザー名（username）の部分一致検索（PostgreSQL `ILIKE`）

**処理フロー**

1. ヘッダーの検索アイコンまたはユーザー検索画面を開く
2. 検索欄にキーワードを入力
3. 入力停止後（デバウンス約300ms）に自動で `GET /api/v1/users?q={keyword}` を送信
4. バックエンドで `username ILIKE '%{keyword}%'` でユーザーを検索
5. 一致したユーザーを一覧表示（アイコン・ユーザー名・フォローボタン）
6. フォローボタンからその場でフォロー可能

**検索仕様**

| 項目 | 仕様 |
|------|------|
| 検索対象 | ユーザー名（username）のみ |
| マッチング | 部分一致（大文字・小文字を区別しない） |
| 自分自身 | 検索結果に表示しない |
| 最大件数 | 20件（ページネーションなし） |
| 空欄・空白のみ | 結果を表示しない |

---

### フォローへの導線

ユーザーを探してフォローする経路は以下の3通り：

| 経路 | 操作 |
|------|------|
| ユーザー検索 | 検索画面でユーザー名を入力 → 検索結果のフォローボタンをクリック |
| タイムラインから | タイムラインの投稿カードのユーザー名クリック → プロフィール画面のフォローボタンをクリック |
| 投稿詳細から | 投稿詳細画面の投稿者・コメント投稿者のユーザー名クリック → プロフィール画面のフォローボタンをクリック |

---

## APIエンドポイント

| メソッド | エンドポイント | 認証 | 説明 |
|---------|--------------|------|------|
| GET | `/api/v1/users?q={keyword}` | 必要 | ユーザー名部分一致検索（最大20件） |

**レスポンス例**

```json
[
  {
    "id": 1,
    "username": "taro_yamada",
    "avatar_url": "https://s3.ap-northeast-1.amazonaws.com/...",
    "is_following": false
  },
  {
    "id": 2,
    "username": "hanako_tanaka",
    "avatar_url": null,
    "is_following": true
  }
]
```
