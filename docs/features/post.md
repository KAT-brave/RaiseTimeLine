# 機能定義書：投稿

## 概要

ログイン済みユーザーがテキスト（最大280文字）と画像（最大4枚）で投稿を作成・削除できる。画像はAWS S3にpresigned URL方式でアップロードする。

---

## 機能一覧

| # | 機能 | 説明 |
|---|------|------|
| 1 | 投稿作成 | テキスト（必須・最大280文字）＋画像（任意・最大4枚）で投稿する |
| 2 | 画像添付 | 投稿作成時に画像を最大4枚まで添付できる |
| 3 | 画像プレビュー | 選択した画像を投稿前にサムネイルでプレビュー表示する |
| 4 | 投稿削除 | 自分の投稿のみ削除できる |
| 5 | 投稿詳細表示 | 投稿内容・添付画像・コメント一覧・いいね数を表示する |

---

## 詳細仕様

### 投稿作成

**入力項目**

| フィールド | 型 | バリデーション |
|-----------|-----|--------------|
| 投稿本文 | 文字列 | 必須・空白のみ不可・280文字以内 |
| 添付画像 | ファイル（複数） | 任意・最大4枚・JPEG/PNG/GIF・1枚10MB以内 |

**処理フロー**

1. ユーザーが投稿作成画面でテキストを入力・画像を選択
2. 残り文字数をリアルタイムでカウント表示（280 - 入力文字数）
3. 投稿ボタンクリック時、フロントエンドでバリデーション
4. 画像がある場合、`POST /api/v1/posts/presigned_url` でS3 presigned URLを取得
5. 取得したpresigned URLへ画像を直接PUTアップロード（S3へ直接転送）
6. `POST /api/v1/posts` で投稿本文とS3画像URLを送信
7. バックエンドがpostsテーブルとpost_imagesテーブルに保存
8. タイムライン画面に戻り、新投稿を先頭に表示

---

### 画像アップロード（S3 presigned URL方式）

```
1. クライアント → POST /api/v1/posts/presigned_url（ファイル名・枚数を送信）
2. Spring Boot → AWS SDKでS3 presigned URLを発行
3. クライアント → presigned URLへ直接PUTアップロード（S3へ直接）
4. アップロード完了後、S3のURLをリスト化
5. POST /api/v1/posts へ本文 + S3 URLリストを送信
6. バックエンド → post_imagesテーブルにURL・表示順を保存
```

**S3バケット・パス**

| 用途 | バケット | パス例 |
|------|---------|--------|
| 投稿画像 | post-images | `post-images/posts/{post_id}/{uuid}.jpg` |

**presigned_urlリクエストボディ例**

```json
{
  "files": [
    { "filename": "photo1.jpg", "content_type": "image/jpeg" },
    { "filename": "photo2.png", "content_type": "image/png" }
  ]
}
```

**画像制約**

| 項目 | 制約 |
|------|------|
| 枚数 | 1投稿につき最大4枚 |
| ファイル形式 | JPEG / PNG / GIF |
| ファイルサイズ | 1枚あたり最大10MB |
| 表示順 | 添付した順番（position: 1〜4）で保存・表示 |

---

### 投稿削除

- 自分が作成した投稿のみ削除ボタンを表示
- 削除前に確認ダイアログを表示
- `DELETE /api/v1/posts/:id` を送信
- バックエンドで投稿者とログインユーザーを照合し、一致しない場合は `403 Forbidden` を返す
- 削除成功後、S3上の添付画像も同時削除（AWS SDK使用）

---

### 投稿詳細表示

- 投稿本文（全文）
- 添付画像（最大4枚、S3からロード）
- いいね数・いいねボタン
- コメント一覧
- 投稿者のアイコン・ユーザー名（クリックでプロフィール画面へ遷移）

---

## APIエンドポイント

| メソッド | エンドポイント | 認証 | 説明 |
|---------|--------------|------|------|
| GET | `/api/v1/posts/:id` | 必要 | 投稿詳細取得 |
| POST | `/api/v1/posts` | 必要 | 投稿作成 |
| DELETE | `/api/v1/posts/:id` | 必要（投稿者のみ） | 投稿削除 |
| POST | `/api/v1/posts/presigned_url` | 必要 | S3 presigned URL発行 |

---

## エラーケース

| ケース | HTTPステータス | エラーメッセージ例 |
|--------|-------------|----------------|
| 本文が空 | 422 | 投稿内容を入力してください |
| 280文字超過 | 422 | 投稿は280文字以内で入力してください |
| 画像5枚以上 | 422 | 画像は最大4枚まで添付できます |
| 非対応ファイル形式 | 422 | JPEG/PNG/GIF形式の画像のみ添付できます |
| ファイルサイズ超過 | 422 | 画像のサイズは10MB以下にしてください |
| 他人の投稿を削除 | 403 | この操作は許可されていません |
| 投稿が存在しない | 404 | 投稿が見つかりません |
| S3アップロード失敗 | 500 | 画像のアップロードに失敗しました |
