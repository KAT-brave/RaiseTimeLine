# 機能定義書：タイムライン

## 概要

タイムラインには2種類ある。全ユーザーの投稿を新着順で表示する**全体タイムライン**と、フォロー中のユーザーの投稿のみ表示する**フォロータイムライン**。タブUIで切り替えられる。

---

## 機能一覧

| # | 機能 | 説明 |
|---|------|------|
| 1 | 全体タイムライン表示 | 全ユーザーの投稿を新着順で一覧表示する |
| 2 | フォロータイムライン表示 | フォロー中のユーザーの投稿のみ新着順で表示する |
| 3 | タイムライン切り替え | タブUIで全体／フォローを切り替えられる |

---

## 詳細仕様

### タイムラインの種類

| 種類 | 表示対象 | APIパラメータ | デフォルト |
|------|---------|-------------|-----------|
| 全体タイムライン | 全ユーザーの投稿 | `GET /api/v1/posts?type=global` | ✓ |
| フォロータイムライン | フォロー中ユーザーの投稿のみ | `GET /api/v1/posts?type=following` | |

- どちらも `created_at` の降順（新着順）で表示
- フォロータイムラインでフォロー中ユーザーが0人の場合は「まだ投稿がありません。ユーザーをフォローしてみましょう。」を表示

---

### 各投稿カードに表示する情報

| 要素 | 説明 |
|------|------|
| 投稿者アイコン・ユーザー名 | クリックでプロフィール画面へ遷移 |
| 投稿日時 | |
| 投稿本文 | 最大280文字 |
| 添付画像サムネイル | 最大4枚 |
| いいねボタン（トグル） | カード上でいいね可能。いいね済みは色変更 |
| いいね数 | |
| コメント数 | クリックで投稿詳細画面へ遷移 |

---

## APIエンドポイント

| メソッド | エンドポイント | 認証 | 説明 |
|---------|--------------|------|------|
| GET | `/api/v1/posts?type=global` | 必要 | 全体タイムライン（全ユーザーの投稿・新着順） |
| GET | `/api/v1/posts?type=following` | 必要 | フォロータイムライン（フォロー中のみ・新着順） |
